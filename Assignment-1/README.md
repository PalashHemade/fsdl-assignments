# HTML Text Formatting Reference

A clean, responsive single-page reference list for HTML text formatting elements. It displays various text formatting tags in a structured table, providing standard descriptions and visual example outputs for each.

## 📝 Description
This assignment acts as a quick reference sheet for basic HTML elements used to format text on web pages. It lists each formatting tag along with its description and a live rendering of its visual style to demonstrate browser defaults.

## 💻 Tech Stack
- **HTML5**: Defines the structural table and displays the text elements.
- **CSS3 (Embedded)**: Simple custom styling for layout structure, table borders, zebra-striping, and hover highlight effects.

## ⚡ Key Features
- **Visual Examples**: Displays live inline HTML elements to preview the direct visual output of each formatting tag.
- **Responsive Table Layout**: Structured table with standard margins, custom background colors, and border styling.
- **Interactivity**: Highlight-on-hover effect applied to rows (`tr:hover`) for readability.
- **Code Highlights**: Custom styling for technical tags using `<code>` and `<pre>` elements.

## 🏷️ Tags Covered
The reference table covers the following elements:
- `<b>` (Bold)
- `<strong>` (Important Text)
- `<i>` (Italic)
- `<em>` (Emphasized Text)
- `<u>` (Underlined Text)
- `<mark>` (Highlighted/Marked Text)
- `<small>` (Smaller Text)
- `<del>` (Deleted/Strikethrough Text)
- `<ins>` (Inserted Text)
- `<sub>` (Subscript Text)
- `<sup>` (Superscript Text)
