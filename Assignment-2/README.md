## 📝 Description
This portfolio is a personal showcase website for **Palash Hemade**, a computer science student and full-stack developer. The layout is optimized to display key sections (About Me, Skills, Experience, Projects, Contact) within a single-page structure.

## 💻 Tech Stack
- **HTML5**: Structured semantic web page using HTML5.
- **CSS3 (Custom)**: Implements visual style tokens, responsive media queries, custom layouts, and glowing micro-animations.
- **JavaScript (Vanilla ES6)**: Orchestrates page behaviors, including custom animations and navigation helpers.
- **Bootstrap 5 (via CDN)**: Provides layout grids, helper utility classes, and the collapsing navigation menu component.
- **Font Awesome 6 (via CDN)**: Embeds clean vector icons throughout the interface.
- **Google Fonts (Poppins)**: Custom typography integration.

## ⚡ Key Features
- **Dynamic Typing Effect**: Re-types titles ("Web Apps.", "Gen AI Solutions.", "Green Hydrogen Tech.") in the hero section with deletion and typing delays.
- **Z-Index Particles**: Interactive particle background styles that enrich visual aesthetics.
- **Scroll Spy**: Auto-highlights the active navigation link corresponding to the scroll location.
- **Glassmorphism Nav**: Navigation header with backdrop blurs (`backdrop-filter: blur`) that applies a subtle shadow when scrolling.
- **Float Animations**: Floating styling adjustments applied to the central code icon using CSS `@keyframes`.
- **Responsive Collapse**: Automatically collapses the hamburger dropdown when a menu selection is clicked on mobile viewports.

## 🎨 Design Details
- **Primary Background**: `#0a0a0a` (Deep black)
- **Secondary Background**: `#111111` (Matte black)
- **Accent Theme Color**: `#0aff9d` (Neon Mint Green)
- **Visuals**: Modern card outlines using semi-transparent whites `rgba(255,255,255,0.05)` with thin boundaries (`1px solid rgba(255,255,255,0.1)`).
